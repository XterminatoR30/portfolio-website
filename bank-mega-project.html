Complete System Workflow

End-to-end pipeline processing from input to multi-cloud distribution

STEP 1

Input Processing

Parse theme, category, subcategory → Load MAIN TRAITS configuration → Extract reference images from ZIP (if provided) → Validate parameters and API credentials

STEP 2

Automated Prompt Generation

Automatic Mode: Call AI with MAIN TRAITS → Generate N variations in batches of 3-4 → Post-process to continuous paragraphs
Manual Mode: Use provided prompt → Apply mandatory prefix if configured

STEP 3

Multi-Provider AI Generation

Select AI provider (Nano-Banana/Imagen-4/Custom LoRA) → Round-robin region selection (4 GCP regions) → Call API with exponential backoff retry → Handle rate limits (429 errors) → Extract and decode base64 image → Save to temp directory

STEP 4

Background Removal & Subject Optimization

Call selected provider (PhotoRoom/Rembg/BIREFNET/SAM2) → Remove background to transparent PNG → Detect subject boundaries via alpha channel → Crop to subject bounding box → Calculate optimal scale maintaining aspect ratio → Resize to maximize canvas usage

STEP 5

Card Template Application (Optional)

Load brand card template PNG → Calculate optimal subject placement in whitespace → Center subject within target area → Apply mathematically perfect rounded corners → Add drop shadow with blur → Save as _card.png variant

STEP 6

Smart File Organization

Generate sequential numbers (TTCCCNNNNN format) → Ensure unique identifiers → Create clean base filenames → Organize into theme/category/subcategory folders → Generate multiple variants (original.png, _nobg.png, _card.png)

STEP 7

Excel Metadata Generation

Create Excel workbook → Add metadata columns (filename, prompt, timestamp) → Embed 150x150px image thumbnails → Format cells with alignment, borders, colors → Save with GMT+7 timestamp for tracking

STEP 8

Multi-Cloud Distribution

Google Drive: Create hierarchical folder structure → Upload all variants → Upload Excel metadata → Upload base64 files to dedicated folder
AWS S3: Upload to S3 bucket → Set appropriate ACL permissions → Enable backup redundancy

FINAL STEP

Output Delivery & System Cleanup

Create comprehensive ZIP file with all variants → Return file paths and cloud URLs → Update tracking JSON files for auditing → Log completion statistics → Clean up temporary files → Process complete ✓
